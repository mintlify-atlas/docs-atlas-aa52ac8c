**VibeVoice-Realtime-0.5B** is a lightweight real-time text-to-speech model supporting streaming text input and robust long-form speech generation. It produces initial audible speech in **~200 milliseconds** (hardware dependent), making it ideal for LLM-to-speech pipelines where output begins before the full response is generated.

## Key specifications

| Spec | Value |
|------|-------|
| Parameters | 0.5B |
| First-audio latency | ~200ms (hardware dependent) |
| Context window | 8K tokens (~10 minutes audio) |
| Speakers | Single speaker |
| Primary language | English |
| Experimental languages | German, French, Italian, Japanese, Korean, Dutch, Polish, Portuguese, Spanish |

## Use cases

- **LLM-to-speech pipelines** — start speaking from the first LLM tokens, long before the full answer is generated
- **Real-time TTS services** — low-latency voice generation for conversational AI
- **Live data narration** — stream and speak dynamic content as it arrives
- **Accessible interfaces** — real-time audio output for screen readers and assistive tools

<Note>
  The streaming model supports **single speaker only**. For multi-speaker conversational generation with up to 4 speakers, see [VibeVoice-TTS](/tts/overview).
</Note>

## Architecture

VibeVoice-Realtime uses an **interleaved windowed design** that differs from the full VibeVoice-TTS model:

- Incrementally encodes incoming text chunks
- Continues diffusion-based acoustic latent generation from prior context in parallel
- Uses **acoustic tokenizer only** (no semantic tokenizer) for lower latency
- Acoustic tokenizer frame rate: **7.5 Hz** (1 frame per ~133ms)
- Base LLM: Qwen2.5-0.5B

See [TTS Architecture](/tts/architecture) for a full deep-dive.

## Multilingual experimental support

Although the model is primarily built for English, it exhibits multilingual capability in 9 additional languages:

**German, French, Italian, Japanese, Korean, Dutch, Polish, Portuguese, Spanish**

<Warning>
  Multilingual behaviors have **not been extensively tested**. Use with caution and share observations with the community. Download experimental voices with `bash demo/download_experimental_voices.sh`.
</Warning>

## Limitations

- Single speaker only (no multi-speaker support)
- English primary — other languages may produce unpredictable outputs
- Voice customization not available without contacting the team
- Very short inputs (3 words or fewer) may produce unstable output
- Does not handle code, mathematical formulas, or uncommon symbols
- Not recommended for commercial use without further testing
